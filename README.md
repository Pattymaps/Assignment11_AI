# Assignment11_AI
# Image Classification Using Random Forest

## Project Overview
This project demonstrates image classification using classical machine learning techniques, specifically the **Random Forest Classifier**. The goal is to build a model capable of accurately classifying images into predefined categories by extracting and learning patterns from pixel data.

The project covers the complete machine learning pipeline, including data preprocessing, model training, hyperparameter tuning, evaluation, feature importance analysis, model comparison with Support Vector Machines (SVM), and a conceptual discussion on deployment.

---

## Dataset
A publicly available **image classification dataset** was used (e.g., a food image dataset such as Pizza vs Burger or a subset of Food‑101).
